Notas importantes para as imagens:
1. Todas as imagens devem ser otimizadas para web
2. Tamanhos recomendados:
   - Banners: 1920x1080px
   - Imagens de produtos: 800x600px
   - Thumbnails: 400x300px
   - Fotos de equipe: 400x400px (quadrado)
   - Fotos de depoimentos: 100x100px (redondas)
3. Formatos recomendados:
   - Para fotografias: JPG
   - Para logos e elementos gráficos: SVG ou PNG
   - Favicon: ICO
4. Tenha certeza de que todas as imagens são de alta qualidade
   e representam bem seus produtos
